Создать плейбук для установки PostgreSQL 14 не на хосте управления.

1.Установить пакет postgesql14-server
2.Выполнить инициализацию БД с использованием  модуля command
	command: /usr/pgsql-14/bin/postgresql-14-setup initdb
3.Изменить порт для приёма входящих подключений на 5433
4.Создать каталог /var/lib/pgsql/14/data/conf.d для собственных настроек.
5.установить в конфигурационном файле postgresql include_dir = 'conf.d'
6.Разрешить доступ по сети на уровне сетевого экрана и конфигурации pg_hba.cfg с любого хоста.
7.каталог с логами перенести в /data/logs/pgsql/14/, исправить месторасположение каталога журналов в конфигурационном файле.
8.Активировать сервис.
